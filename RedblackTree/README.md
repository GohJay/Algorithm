# Red-black Tree Builder

## Overview
레드블랙트리 알고리즘을 시각화한 프로그램 <br>
레드블랙트리란 자가 균형 이진 탐색 트리(self-balancing binary search tree)로서, 대표적으로는 연관 배열 등을 구현하는 데 쓰이는 자료구조이다.
